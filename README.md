# Http Client Library For C++

## 概要

Http 送信を実装したクライアントライブラリ。

## Compile

```
$ cd src
$ sudo gmake
```

## Development

```
cat yum-devel.txt|xargs -I@ sh -c "sudo yum install -y @"
```

## Test

```
$ cd src
$ gmake test
```
